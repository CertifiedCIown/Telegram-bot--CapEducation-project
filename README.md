# "ОБО МНЕ"-CapEducation-project
(ENG)This is a Python file with code based on "telebot", which is a Telegram-bot-portfolio. (RUS)Это файл на Python с кодом, основанным на "telebot", который является Telegram-ботом-портфолио
## Возможности
-Раздел «Обо мне»
-Моя цель
-Как я пришёл в IT
-Мой ментор
-Точка А → Точка Б
-Хобби и интересы
-Лучшие проекты
-Навигация с помощью команд и Inline-кнопок
-Поиск команд в сообщении с использованием регулярных выражений (`re`)
-Хранение контента в отдельном JSON-файле
-Поддержка аргументов командной строки (`sys.argv`)
-Обработка возможных ошибок (`try-except`)

## Используемые технологии
-Python 3
-pyTelegramBotAPI
-JSON
-Regular Expressions (`re`)
-pathlib
-sys.argv

## Установка
Установите зависимости через:
bash
pip install -r requirements.txt
Или установите вручную, список компонентов записан в requirements.txt

##Запуск бота
Перед запуском ОБЯЗАТЕЛЬНО:
Проверьте, находятся ли файлы: MainText.json и main.py в одной папке, а также наличие требуемых библиотек.
Затем получите свой токен бота от BotFather из telegram и затем вставьте его заместо слова TOKEN(без кавычек) в команду:
python main.py TOKEN
После этого вставьте команду в терминал и запустите команду
